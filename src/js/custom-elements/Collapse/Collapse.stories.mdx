Вложенные компоненты:

-   Toggler

Обозначается атрибутом "data-collapse-toggler".

Пример разметки:

```html
<button data-collapse-toggler>Toggler</button>
```

-   Toggler text

Применяется в случае, когда надо менять текст тогглера. Обозначается атрибутом "data-collapse-toggler-text".

Пример разметки:

```html
<button data-collapse-toggler data-collapse-collapsed-text="Collapse" data-collapse-expanded-text="Expand">
    <span data-collapse-toggler-text>Expand</span>
</button>
```

-   Content

Обозначается атрибутом "data-collapse-content".

Пример разметки:

```html
<div data-collapse-content>Content here</div>
```

Под капотом используется ResizeObserver для перерасчета высоты контента при динамическом изменении (например, когда изображение загрузилось после инициализации компонента).
